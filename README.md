# 🦷 SmileCare - Modern Dental Clinic Landing Page

A professional, high-performance, and responsive landing page designed for dental clinics and medical centers.

## ✨ Features
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices.
* **Modern UI/UX:** Clean design with a focus on patient trust and easy navigation.
* **Interactive Elements:** Smooth scroll animations, hover effects, and a functional contact form.
* **Performance:** Lightweight and fast loading times using clean HTML/CSS/JS.
